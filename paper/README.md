# quantmsdiann manuscript (LaTeX)

First draft of the **quantmsdiann** software paper, formatted for
[Molecular & Cellular Proteomics (MCP)](https://www.sciencedirect.com/journal/molecular-and-cellular-proteomics)
using Elsevier's `elsarticle` class.

## Layout

```
paper/
  main.tex              # master document (abstract + body; no GA/highlights)
  graphical_abstract.tex  # graphical abstract + research highlights (separate upload)
  macros.tex            # \placeholder{}, \quantmsdiann, etc.
  mcpfigures.sty        # MCP column widths (3.5 / 5 / 7.2 in; max height 6 in)
  references.bib        # bibliography (numeric elsarticle-num style)
  sections/
    introduction.tex    # Background section body
    results.tex
    discussion.tex
    conclusions.tex
    methods.tex
    declarations.tex
    figures.tex         # figure floats + links to analysis/figures SVGs
  figures/pdf/          # PDF exports for LaTeX (generated; git-ignored)
  Makefile
```

Manuscript figures in `sections/figures.tex` follow MCP guidelines:

| Width macro | Size | MCP use |
|---|---|---|
| `\mcponecolwidth` | 3.5 in | single-column panels |
| `\mcponehalfcolwidth` | 5.0 in | 1.5-column panels |
| `\mcpdoublecolwidth` | 7.2 in | full-page composite figures |

Composite figures (Figs. 1–5 in the draft) are marked **pending** until
multi-panel SVGs are assembled. **Fig. 1** is generated by
[`analysis/figure_f1_architecture_scaling.py`](../analysis/figure_f1_architecture_scaling.py)
into `analysis/figures/manuscript/fig1_architecture_scaling.svg`.

## Build

```bash
cd paper
make pdf                  # main.pdf + graphical_abstract.pdf
make graphical-abstract   # graphical_abstract.pdf only
make figures              # convert linked SVGs to PDF
make all                  # figures + both PDFs

# Regenerate Fig. 1 composite from repo root:
python -m analysis.figure_f1_architecture_scaling
```

Requirements: TeX Live with `elsarticle`, `latexmk`, `bibtex`.

Draft placeholders appear as italic bracketed text via `\placeholder{...}`.
Replace these with measured values before submission.

## Submission notes (MCP)

- Initial submission: content completeness matters more than strict MCP layout.
- Revised manuscripts: adopt MCP figure sizing, 300 dpi minimum, Helvetica labels.
- LaTeX may use double-column layout (`\documentclass[final,3p,times,twocolumn]{elsarticle}`).
- Upload all `.tex`, `.bib`, `.bst`, and figure files in a **flat directory** for Editorial Manager.
- Graphical abstract and research highlights live in `graphical_abstract.tex` for separate submission.
